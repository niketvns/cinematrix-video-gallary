## Video Library App

This is an video gallary app.

## Features:
1. Landing Page
2. Video Details Page
3. Authentication
    1. Login/Logout
    2. Signup
4. Liked Video Page
5. Watch Later Page
6. History Page
7. Loading & Alerts
8. Responsiveness

## Tech Stack Used
- React.js
- Tailwind CSS
- MockBee (mock backend)
